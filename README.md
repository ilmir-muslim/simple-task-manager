# Simple Task Manager

Простой консольный менеджер задач для практики работы с Git.

## Функциональность
- Добавление задач
- Просмотр списка задач
- Отметка задач как выполненные
- Удаление задач

## Установка
```bash
git clone https://github.com/ваш-username/simple-task-manager.git
cd simple-task-manager
```
## Разработка
Для добавления новой функциональности:
1. Создайте ветку от main
2. Реализуйте feature
3. Сделайте PR в main
